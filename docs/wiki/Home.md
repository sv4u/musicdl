# musicdl Wiki

Welcome to the musicdl wiki. musicdl is a **CLI-only** tool: use `musicdl plan <config>` and `musicdl download <config>` to generate and run download plans. There is no web UI or API. This wiki contains detailed documentation for architecture, deployment, and development.

## Pages

- **[Architecture](Architecture.md)**: Detailed architecture documentation, plan-based system, and component descriptions
- **[CI/CD](CI-CD.md)**: GitHub Actions workflows, release process, and dependency management
- **[TrueNAS Deployment](TrueNAS-Deployment.md)**: Complete guide for deploying musicdl on TrueNAS Scale
- **[Development](Development.md)**: Development setup, testing guidelines, and contributing
